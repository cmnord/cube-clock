# Cube clock

![Progress pic](progress.jpg)

## Parts

- ESP32 devkit v1 ([pinout][esp32pinout])
- 8 220 kOhm resistors
- seven-segment display (5461AS)

## Software dependencies

- Arduino
- ESP32 dev module support for Arduino
- [SevSeg][sevseg] library
